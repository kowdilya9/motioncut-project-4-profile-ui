# MotionCut Project 4 - Profile Page UI

Mobile Profile Page UI design created using Figma as part of the MotionCut UI/UX Internship.

## 📱 Mobile Screen Preview
![Profile UI Design](iPhone%2014%20%26%2015%20Pro%20-%201.png.png)

## 🎯 Features Designed
- **Header Section:** Profile title and back arrow navigation
- **Profile Info:** User avatar, KOWDILYA, UI/UX Designer, Edit Profile button
- **Stats Section:** 12 posts | 14k followers | 320 following
- **Settings List Cards:** Account Settings, Notifications, Privacy, Help & Support, Logout

## 🛠️ Tools Used
- **Tool:** Figma
- **Fonts:** Sans-serif / Serif mix
